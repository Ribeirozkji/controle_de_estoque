# Especificação do Aplicativo Android

## Objetivo
Transformar o sistema atual de controle de estoque em uma plataforma Android completa com estoque, vendas, comandas digitais, clientes, relatórios e IA.

## Módulos
- Dashboard
- Produtos
- Categorias
- Fornecedores
- Estoque
- Movimentações
- Clientes
- Comandas Digitais
- Mesas
- Relatórios
- Usuários
- Configurações

## Comanda Digital
- Abrir comanda
- Associar mesa
- Associar cliente
- Adicionar itens consumidos
- Remover itens
- Alterar quantidades
- Fechar comanda
- Registrar pagamento
- Histórico completo

## Controle Automático de Estoque
Toda venda deve gerar baixa automática de estoque.

## Ficha Técnica
Cada produto vendido deve consumir ingredientes automaticamente.

## IA
- Assistente de gestão
- Previsão de estoque
- Sugestão de reposição
- Insights de vendas
- Produtos mais vendidos
- Produtos parados

## Banco de Dados
Migrar de JSON para MySQL.

## Backend
Node.js + Express.

## Frontend
Aplicativo Android moderno com Material Design 3.
